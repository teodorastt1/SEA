# 🥤 Питиетата (Bevera)

## 📌 Описание

Уеб приложение за управление на склад и продажба на напитки. Проектът е разработен като дипломен проект и включва пълна система за работа с продукти, поръчки, потребители и дистрибутори.

---

## 🎯 Основна цел

Създаване на цялостна система, която:

* управлява наличности (StockQty)
* позволява поръчки от клиенти
* обработва поръчки от служители
* поддържа връзка с дистрибутори
* следи приходи и разходи

---

## 👥 Роли в системата

* **Админ** – пълен контрол (продукти, категории, потребители, финанси)
* **Служител (Worker)** – обработва поръчки и следи склад
* **Клиент (Client)** – разглежда продукти и поръчва
* **Дистрибутор (Distributor)** – изпълнява заявки за зареждане

---

## ⚙️ Основни функционалности

* CRUD за продукти, категории и марки
* Регистрация и вход с роли

## ▶️ Стартиране

1. Отворете проекта във Visual Studio
2. Настройте `appsettings.json`
3. Изпълнете:

```
Update-Database
```

4. Стартирайте проекта
